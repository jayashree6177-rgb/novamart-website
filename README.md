# NovaMart — Multi-Vendor E-Commerce Marketplace UI/UX

A front-end website built from a Figma design, converted into working HTML, CSS, and JavaScript.

## Pages
- **Home** (`index.html`) — hero banner, latest arrivals, flash deals
- **Category** (`category.html`) — category grid, new launches, brand spotlight
- **Product** (`product.html`) — image gallery, size selector, seller info

## How to run locally
1. Clone this repo
2. Open `index.html` in a browser, or use VS Code's Live Server extension
